Ingress Intel Total Conversion - Community Edition (IITC-CE) es un complemento de navegador web para el juego de Realidad Aumentada(RA) Ingress, más rápido que el mapa estándar de Ingress, y ofrece muchas más opciones.

Disponible en versión de escritorio para navegadores web como Firefox, Chrome, entre otros; y como app para smartphones.

Página web: https://iitc.app
Repositorio GitHub: https://github.com/IITC-CE/ingress-intel-total-conversion/
