Ingress Intel Total Conversion - Community Edition (IITC-CE) ist eine Browser-basierte Erweiterung für das virtuelle Spiel Ingress. Sie ist schneller als die Browser-basierte Intel Map von Ingress und bietet viel mehr Funktionen.

Sie ist verfügbar für Destop-Browser wie Firefox und Chrome und als mobile App.

Webseite: https://iitc.app
GitHub Repository: https://github.com/IITC-CE/ingress-intel-total-conversion/
